Auto-approve Codex permission prompts

v2.1

Que hace

Codex se para constantemente a pedir permisos. CodexBot detecta el badge En espera de aprobacion en el sidebar y pulsa "2" automaticamente para aprobar.

🟢
Deteccion por badge verde
Escanea el sidebar buscando el badge verde. 0 falsos positivos.
🖱️
Pausa si usas el raton
No actua si has movido el raton en los ultimos 5 segundos.
⏱️
Configurable
Duracion, intervalo y modo sin screenshots ajustables.
💻
Windows + macOS
PowerShell para Windows, Python para Mac.

Como funciona

1
Cada 6 segundos captura un screenshot de la ventana de Codex
2
Escanea el sidebar izquierdo buscando pixeles verdes del badge
3
Si detecta 100+ pixeles verdes y el raton lleva 5s quieto, trae Codex al frente
4
Envia "2" (aprobar y no preguntar mas) y devuelve el foco

Arrancar

1. Clonar el repo (solo la primera vez):

PowerShellgit clone https://github.com/csilvasantin/CodexBot.git cd CodexBot

2. Lanzar (abre ventana propia):

PowerShellpwsh -File Launch.ps1

O directamente (en la misma terminal):

PowerShellpwsh -File CodexBot.ps1

Con opciones:

PowerShell# 60 minutos, check cada 4 segundos pwsh -File CodexBot.ps1 -Duration 60 -Interval 4

Parar: pulsa q en la ventana de CodexBot.

1. Instalar dependencias (solo la primera vez):

Terminalpip3 install Pillow pyautogui git clone https://github.com/csilvasantin/CodexBot.git cd CodexBot

2. Dar permisos (solo la primera vez):

Ve a Ajustes del Sistema > Privacidad > Accesibilidad y Grabacion de Pantalla — anade Terminal (o iTerm).

3. Lanzar:

Terminalpython3 codexbot_mac.py

Con opciones:

Terminal# 60 minutos, check cada 4 segundos python3 codexbot_mac.py -d 60 -i 4

Parar: Ctrl+C
