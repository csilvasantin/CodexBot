# CodexBot

## 1 2 3 basico

1. leer `onboarding` antes de cambiar el bot;
2. dejar el bot funcionando o documentar claramente el bloqueo real;
3. cerrar con HTML y URL publica verificable.

## Objetivo

CodexBot automatiza interacciones repetitivas relacionadas con Codex y sirve como apoyo operativo para reducir aprobaciones manuales.

## Regla de trabajo

1. comprobar primero el estado actual del bot;
2. hacer cambios concretos y verificables;
3. documentar el resultado para continuidad por otra IA.

## Regla de cierre

Al terminar trabajo relevante:

1. generar una salida final en HTML;
2. entregar una URL publica comprobable;
3. si se usa `Nomeacuerd0`, tratarlo solo como filtro ligero y no como seguridad fuerte.
